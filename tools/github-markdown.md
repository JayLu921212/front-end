# Markdown[github版本]

## 概述

Markdown是一种轻量级的标记语言，实际上是html的超集，我们可以在Markdown中自由的使用html语法。github上的README就是用的markdown的语法。

*markdown写成的文档由于引入的css文件不同，呈现出来的样式也有所不同，在不同的markdown工具编写markdown，生成文档的有所不同。*
